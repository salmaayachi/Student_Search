# StudentSearch

A .NET Core console application to manage and search student data using LINQ.

## Features

- Search for adult students (age >= 18).
- Search for students by major.
- Sort students by age.
- Perform global searches on all fields.
- Serialize student data to JSON and XML formats.

## Requirements

- .NET Core SDK
- Newtonsoft.Json package
